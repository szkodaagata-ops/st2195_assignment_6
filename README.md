# ST2195 Assignment 6

This repository contains the sixth assignment for the course **Programming for Data Science ST2195**.

The data used in this assignment comes from the **European Central Bank (ECB)**.  
Two datasets are included:  
- **speeches** – a dataset containing ECB speeches, available at: https://www.ecb.europa.eu/press/key/shared/data/all_ECB_speeches.csv  
- **fx** – daily EUR/USD exchange rate data, available at: https://data.ecb.europa.eu/data/datasets/EXR
